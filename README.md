#Attrition Analytics Dashboard

An interactive **HR Analytics Dashboard** built in **Power BI** to analyze employee attrition, workforce demographics, salary distribution, and job-role trends. This dashboard provides actionable insights to help organizations understand employee turnover patterns and improve workforce decision-making.

## Tools & Technologies Used

* Microsoft Power BI
* Power Query
* CSV Dataset
* DAX (Data Analysis Expressions)

## Key Features

* Employee Attrition Analysis
* Attrition Rate KPI Tracking
* Department-wise Filtering
* Attrition by Education Field
* Attrition by Age Group
* Attrition by Job Role
* Salary Slab Analysis
* Years at Company Trend Analysis
* Interactive Visualizations & Slicers

## Dashboard Insights

* Total Employees: **1.47K**
* Attrition Count: **238**
* Attrition Rate: **16.1%**
* Average Salary: **6.50K**
* Average Age: **36.92**
* Average Years at Company: **7.01**

## Data Preparation

* Cleaned and transformed raw CSV data using **Power Query**
* Created calculated measures using **DAX**
* Built interactive KPIs and dynamic visuals in Power BI
